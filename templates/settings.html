Threads API セットアップ手順
  1. Meta Developer Portal (developers.facebook.com) でアプリを作成
  2. アプリに「Threads」製品を追加
  3. テストユーザーを追加し、threads_basic + threads_content_publish 権限を付与
  4. アプリの「リダイレクト URI」に {{ settings.callback_url }} を登録
  5. 上の設定に App ID・App Secret・アプリの URL を入力して保存
  6. 「Threads で認証する」ボタンをクリック → ブラウザで認証 → 自動でトークンが保存されます
