Threads 手動認証

コールバック URL が登録されていない場合や自動リダイレクトが機能しない場合に使用してください。

error_code 1349187 について： Meta Developer Portal の「リダイレクト URI」に以下の URL が登録されていないと発生します。
{{ redirect_uri }}

推奨： 上記URLを Meta Developer Portal → アプリ → Threads API → リダイレクト URI に追加すれば自動フローが使えます。
今すぐ使う： 以下の手順で手動コード入力からトークンを取得できます。
Step 1 Meta Developer Portal でリダイレクト URI を登録する（推奨）
  1. developers.facebook.com を開く
  2. 対象アプリ → Threads API → 設定 を開く
  3. 「コールバック URL」または「リダイレクト URI」に以下を追加して保存：
    {{ redirect_uri }}
  4. 登録後は 自動認証フロー を使えます
Step 2 以下の認可 URL をブラウザで開き、Threads アカウントで認証する

以下のボタンをクリック（または URL をコピーしてブラウザで開いてください）

Threads 認証ページを開く
認可 URL を確認する
Step 3 認証後にリダイレクトされた URL から code をコピーして貼り付ける

認証が完了すると、ブラウザが以下のような URL にリダイレクトされます：
{{ redirect_uri }}?code=ここの値をコピー&state=...
ページが表示されなくてもブラウザの URL バーから code= の後の値をコピーできます。

URL の ?code= 以降、&state= より前の文字列を貼り付けてください
